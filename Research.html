Data-driven approach to complex system
    Data-driven methods are useful in real-world applications and in model interpretation.
    Glassy dynamics. The Kuramoto system exhibits various phenomena depending on the network structure and the coupling between oscillators. One interesting phenomenon is the glassy behavior of the order parameter, which emerges when the interaction matrix has high rank. The reason behind this phenomenon remains an open question. Instead, we numerically analyze the low-dimensional structure of this glassy behavior in the Kuramoto system using Extended Dynamic Mode Decomposition (EDMD) and identify a signature of this feature. Furthermore, EDMD can be viewed as a Galerkin approximation of the Koopman operator. We explore the connection between the glassy behavior and the continuum equation, linking it to the Koopman operator of the Kuramoto system.

Mathematical analysis of collective behavior
    In nature, many collective behaviors are observed, and some have been modeled using differential equations. Proving numerical results for such differential equations is challenging but important.
    Inertial Kuramoto model. The Kuramoto model is a well-known synchronization model for \(N\) limit-cycle oscillators with phases \(\theta\), each of which has its own natural frequency \(\nu_i\). When considering an all-to-all network connection, this model is described by the following differential equations: for each \(i=1,\ldots,N\), \[ \label{eq:example} \dot{\theta}_i(t) = \nu_i + \frac{\kappa}{N}\sum_{j=1}^N \sin(\theta_j(t) - \theta_i(t)),\quad t>0, \] where \(\kappa\) is the coupling stength. Here, we consider the inertial effect of each oscillator, and the model is represented by the following system: for each \(i=1,\ldots,N\), \[ \label{eq:example2} m\ddot{\theta}_i(t) + \gamma\dot{\theta}_i(t) = \nu_i + \frac{\kappa}{N}\sum_{j=1}^N \sin(\theta_j(t) - \theta_i(t)),\quad t>0, \] where \(m\) is the inertia and \(\gamma\) is the damping coefficient. This model was developed to describe the synchronization of a specific type of firefly and transient stability in power systems. From numerical simulations, one can easily observe phase-locking in both systems, meaning that \(\exists\lim_{t\to\infty} (\theta_i(t)-\theta_j(t))\) for each \((i,j).\) However, proving this is challenging, especially when each initial phase is randomly distributed on the limit cycle. A recent study provides a proof of this for the first-order system \(\eqref{eq:example}\). In the work I, we present the counterpart of that result for the second-order system \(\eqref{eq:example2}\).
    On the other hand, when \(m\) is large, the second-order system exhibits a distinct featureâ€”an infinite number of phase overlaps on the unit circle (works I,II). This can be regarded as a generalization of the damped pendulum equation since, in the two-oscillator regime, the phase difference is governed by the damped pendulum equation.
